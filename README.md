# The Luxe Bazaar ✨

A full-stack MERN e-commerce web application for fashion, beauty, skincare and lifestyle products.

## Live Demo

Frontend: https://the-luxe-bazaar-shubhangi.vercel.app  
Backend: https://the-luxe-bazaar.onrender.com

## Features

- User authentication
- Product listing
- Search and category filter
- Wishlist
- Cart
- Checkout
- Order placement
- Order history
- User profile
- Admin product management
- Responsive UI
- MongoDB Atlas database
- Frontend deployed on Vercel
- Backend deployed on Render

## Tech Stack

**Frontend**
- React
- Vite
- CSS
- Axios
- React Hot Toast

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt.js
- Multer

## Deployment

- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
